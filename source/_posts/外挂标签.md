## 行内文本样式 text

1. 带 {% u 下划线 %} 的文本
2. 带 {% emp 着重号 %} 的文本
3. 带 {% wavy 波浪线 %} 的文本
4. 带 {% del 删除线 %} 的文本
5. 键盘样式的文本 {% kbd command %} + {% kbd D %}
6. 密码样式的文本：{% psw 这里没有验证码 %}

## 行内文本 span
- 彩色文字
  在一段话中方便插入各种颜色的标签，包括：{% span red, 红色 %}
 {% span yellow, 黄色 %}
 {% span green, 绿色 %}
 {% span cyan, 青色 %}
 {% span blue, 蓝色 %}
 {% span gray, 灰色 %}
- 超大号文字
  文档「开始」页面中的标题部分就是超大号文字。
  {% span center logo large, Volantis %}
  {% span center small, A Wonderful Theme for Hexo %}
